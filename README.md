LinkedIn Job Posts Analysis

This project analyzes 30,000+ LinkedIn job postings to uncover trends in job roles, in-demand skills, hiring industries, and geographic patterns. Using Python for data cleaning and exploratory analysis, and Power BI for interactive visualization, it offers valuable insights for job seekers, recruiters, and workforce analysts.
